# FunnyASR
  
A web UI for ASR based on funasr.  

现在很多手机和电脑端的语音识别 APP 居然要收费，太过分了有木有！那本项目就基于开源的 funasr 进行本地的端到端语音识别，为大家做一个简单的免费语音识别工具。
